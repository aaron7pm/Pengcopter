Cave Copter Version 1.0
by Heiko Nolte, written February 2010
===============================================

This game is distributed under GNU GENERAL PUBLIC LICENSE, Version 3.

To play install Python 2.x and PyGame API.

Mission:
Penetrate the alien cave system. Avoid hostile UFOs, collect ammunition
and refuel your helicopter in order to stay alive.

Controls: 
- Use arrow keys to control helicopter.
- Use space to fire rocket.
